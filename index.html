General Installation Guide

Follow these steps to install VTOP-CLI on your OS

For All Operating Systems

1. Download the appropriate VTOP-CLI file for your operating system.

2. Create a dedicated folder for the VTOP-CLI tool, such as C:\Users\[username]\vtop-cli on Windows, or a similar location on Linux/MacOS.

3. Move the downloaded file to the created folder.

Windows

- Access System Properties and open Environment Variables.

- Edit the 'Path' variable under User variables to include the path to your VTOP-CLI folder.

- Apply changes

Linux/MacOS

- Open a terminal and navigate to the VTOP-CLI folder.

- Modify your shell's profile script to include the VTOP-CLI folder in your PATH environment variable.

- Apply changes by restarting the terminal.

Verify the installation by running vtop-cli in your command line or terminal. The tool should launch successfully.
